## aria2-connection-unlocked
[![Auto-Sync Patch & Build (Windows x64)](https://github.com/tasy5kg/aria2-unlocked/actions/workflows/auto-sync-patch-build-windows-x64.yml/badge.svg)](https://github.com/tasy5kg/aria2-unlocked/actions/workflows/auto-sync-patch-build-windows-x64.yml)

This repository builds [aria2/aria2](https://github.com/aria2/aria2/tree/master) for `Windows x64` daily from the latest source code, with the `max-connection-per-server` limit removed.
- **Daily Sync & Build:** Automatically synced and built from the official `aria2` repository every day.
- **Connection Limit Removed:** The `max-connection-per-server` limit (`16`) is removed at compile time.
- **Avoid Redundant Builds:** Builds will be skipped if there are no upstream updates.

Click [**Releases**](../../releases/latest) to download the latest build of `aria2c.exe`.

## aria2 - 连接数解锁版
[![Auto-Sync Patch & Build (Windows x64)](https://github.com/tasy5kg/aria2-unlocked/actions/workflows/auto-sync-patch-build-windows-x64.yml/badge.svg)](https://github.com/tasy5kg/aria2-unlocked/actions/workflows/auto-sync-patch-build-windows-x64.yml)

本仓库每日基于最新源码构建 `Windows x64` 版 [aria2/aria2](https://github.com/aria2/aria2/tree/master)，并去除其 `max-connection-per-server` 限制。
- **每日同步与构建:** 每天自动与官方 `aria2` 仓库进行同步和构建。
- **去除最大连接数限制:** 在构建时去除 `max-connection-per-server` 上限为 `16` 的限制。
- **避免重复构建:** 如果官方仓库没有更新，则不重复进行构建。

点击 [**Releases**](../../releases/latest) 下载最新构建的 `aria2c.exe`.
